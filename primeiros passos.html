Entendendo comunicação clientexserver

A comunicação cliente se refere a pessoa que utiliza o serviço como acessar o instagram, facebook, YouTube.Por outro lado, a comunicação server se refere a quem produz o site para servir o cliente/ usuário da web. Logo, cliente é sempre o usuário que acessa os serviços criados pelo servidor da máquina.

Introdução ao HTML na prática

A linguagem em HTML é a forma de se criar websites desde um blogue a um site de uma empresa. Essa linguagem é escrita por meio de tags e seu atributos e muitas das vezes são conjugadas com CSS e JavaScript. Os sites antigos só existe o HTML puro, porém os sites mais modernos são incorporados o CSS para dar cor e forma ao site. Uma das vantagens da linguagem html serve tanto para cilent como para servidor.

O que são Tags

As tags são basicamente palavras em ingles que servem de comandos para se formar a página na web. É por meio delas que se pode escrerver e organizar os sites de maneira simples.

O que são atributos

Os atributos são como se fossem uma extensão das tags, uma vez que para determinados comandos é necessário usa-los em conunjunto com as Tags.
